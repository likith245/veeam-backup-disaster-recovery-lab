# Veeam Backup & Disaster Recovery Lab

## Overview

This project demonstrates the implementation of a backup and disaster recovery solution using Veeam Backup & Replication 13 Community Edition in a Windows Server environment.

## Technologies Used

- Veeam Backup & Replication 13
- Windows Server 2022
- VMware Workstation
- PowerShell
- WinRM

## Key Features

- Protection Group deployment
- Veeam Agent installation and management
- Automated server backups
- Backup repository configuration
- Restore point creation
- File-level recovery testing
- Disaster recovery validation

## Lab Outcome

Successfully protected a Windows Server, generated restore points, simulated accidental data loss, and restored deleted data using Veeam File-Level Recovery.
